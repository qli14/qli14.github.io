How to start?

  1. Install Node.js.
  2. Use node -v and npm -v to confirm the success of installation.
  3. In stall Git.exe.
  4. git --version to confirm.
  5. npm install hexo-cli -g
  6. npm install hexo --save
  7. hexo -v to confirm it.
  8. More details can be found here.

How to deploy?

  1. hexo clean
  2. hexo generate
  3. hexo serever
  4. hexo deploy
